# Precision-Psychiatry-data-cleaning-
Data cleaning pipeline I have created at the Center for Precision Psychiatry - Quebec
During my mandate, I used a jupyter notebook combined with the python package Pandas to clean a messy dataset of many thousand variables from psychometric questionnaires, such as the  Mini International Neuropsychiatric Interview (MINI), the Brief Psychiatric Rating Questionnaire (BPRS) and the Hamilton Depression Rating Questionnaire (HAM-D). 
- Transforms Clinician GLobal Impression (CGI) from long to wide format
- Implement conditional formatting logic to remove missing data or variables
- Link questionnaire IDs to alphanumerical numbers to ensure participant's anonymity
